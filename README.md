# Final Project: Intake-to-Note Assistant

This project is a GenAI-powered assistant that transforms intake form text into structured, actionable notes for wellness consults. It demonstrates prompt engineering, context management, and evaluation best practices.

## Setup

1. Clone this repository or copy the files into your workspace.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:
   ```bash
   streamlit run app.py
   ```

See `project_plan.md` for design details and `eval_set.md` for test cases.
